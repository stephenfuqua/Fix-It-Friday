# EdFi.FIF.API

## Description

GraphQL-based API for supporting the Ed-Fi Fix-it-Friday solution.

## Installation

```powershell
yarn install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## TeamCity

Additional commands for TeamCity support

```powershell
yarn test:ci
yarn lint:ci
```
